2396In this library, the numbers are represented interenally in base B = 10**N,
2397where N is the largest possible integer that does not cause overflow in the
2398intermediate computations. The base B elements are stored in an array, with the
2399least significant element stored in array element zero. There are no leading
2400zero elements, except a single zero element when the number is zero. For
2401instance, if B = 10000, the number 1234567890 is represented internally as
